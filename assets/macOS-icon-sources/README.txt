The dist/ folder contains the app icons that should be bundled with Reflection for macOS:

- dist/Reflection.icon is the Apple Icon Composer file for modern macOS. It contains light, dark, and mono variants.
- dist/Reflection-Legacy.iconset is designed for older versions of macOS (pre-Tahoe/pre-Liquid Glass).

The icons were created in Sketch <https://www.sketch.com> using Apple’s official templates.
The source files are in the sketch/ folder.
